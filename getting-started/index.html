    Mengukur Jarak Data¶

    Mengukur jarak data¶

    A. Mengukur Jarak Tipe Numerik¶

    ​ Atribut numery bersifat kuantitatif dan biasanya dalam bentuk tipe integer, dimana memiliki nilai berupa kuantitas yang terukur dan dinyatakan dalam nilai-nilai bulat yang dapat diskalakan secara interval atau rasio.

    Euclidean Distance¶

    ​ Euclidean distance adalah formula untuk mencari jarak antar 2 vektor x,y dimana jarak merupakan hasil jumlah dengan pangkat setengah dari nilai xik dikurangi xij yang dukadratkan dengan batas i=1 sampai batas ke-m. Euclidean distance digunakan untuk penghitungan jarak dalam distance space antar jarak terpendek yang bisa didapatkan antara dua titik yang diperhitungkan. $$ d\left ( x,y \right )= \left ( \sum_{i=1}^{m}\left ( xik-xij \right )^{2} \right )^\frac{1}{2} $$ Keterangan :

    Xik = nilai X pada training data

    Xjk = nilai X pada testing data

    m = batas jumlah banyaknya data

    ​ Untuk Big O dari akar kuadrat adalah O(sqrt(n)). Untuk Big O dari kuadrat adalah O(n2). Untuk Big O dari penjumlahan adalah O(n). Sehingga dapat diketahui bahwa Big O dari Euclidean adalah O(sqrt(n2)). Jika semakin besar hasil nilai dari rumus di atas maka akan semakin jauh tingkat kesamaan antara data training dan data testing, Sebaliknya, jika hasil nilainya semakin kecil maka akan semakin dekat tingkat kesamaan antara data training dan data testing.

    Manhattan Distance¶

    ​ Manhattan Distance atau Taxicab Geometri adalah formula untuk mencari jarak d antar 2 vektor x,y pada ruang dimensi n , dimana jarak merupakan hasil jumlah dari xi dikurangi yi dengan batas i=1 sampai batas ke-m. Manhattan distance sering digunakan karena kemampuannya dalam mendeteksi keadaan khusus seperti keberadaaan outliers dengan lebih baik. $$ d _ { \operatorname { man } } = \sum _ { i = 1 } ^ { n } \left| x _ { i } - y _ { i } \right| $$ Keterangan:

    xi = nilai X pada data latih.

    yi = nilai Y pada data uji.

    n = batas jumlah banyaknya data

    ​ Jika angka hasil dari rumus diatas besar, maka tingkat kemiripan antara kedua objek akan semakin kecil. Sebaliknya, jika angka hasil rumus tersebut kecil, maka tingkat kemiripan antara kedua objek, maka akan semakin besar. Objek yang dimaksud adalah data latih dan data uji yang akan dihitung tingkatkemiripannya.

    Minkowski Distance¶

    ​ Minkowski distance adalah formula pengukuran antar 2 titik pada ruang vektor normal yang merupakan hibridisasi dari mengeneralisasi euclidean distance dan mahattan distance. $$ d _ { \operatorname { min } } = ( \sum _ { i = 1 } ^ { n } | x _ { i } - y _ { i } | ^ { m } ) ^ { \frac { 1 } { m } } , m \geq 1 $$ Keterangan:

    x*i* = nilai x pada data ke-i

    y*i* = nilai y pada data ke-i

    n= batas jumlah banyaknya data

    i=1 = batas data dari i = 1

    m = pangkat bilangan real positif

    Average Distance¶

    ​ Average distance merupakan suatu formula yang dikembangkan untuk menutupi kelemahan dari formula Euclidean distance, dimana average distance merupakan hasil kali jumlah dengan pangkat setengah dari nilai xik dikurangi xij yang dukadratkan dengan batas i=1 sampai batas ke-m dan dari 1/banyaknya fitur. $$ d _ { a v e } = \left ( \frac { 1 } { n } \sum _ { i = 1 } ^ { n } ( x _ { i } - y _ { i } ) ^ { 2 } \right) ^ { \frac { 1 } { 2 } } $$ Keterangan:

    x*ik* = nilai x pada data ke-ik

    y*ij* = nilai y pada data ke-ij

    n= batas jumlah banyaknya data

    i=1 = batas data dari i = 1

    n= banyaknya fitur

    Weighted euclidean distance¶

    ​ Weighted euclidean distance merupakan formula yang digunakan untuk menyempurnakan formula dari euclidean distance, dimana diformulasikan sebagai berikut: $$ d _ { w e } = \left ( \sum _ { i = 1 } ^ { n } w _ { i } ( x _ { i } - y _ { i } \right) ^ { 2 } ) ^ { \frac { 1 } { 2 } } $$ Keterangan:

    x*i* = nilai x pada data ke-i

    y*i* = nilai y pada data ke-i

    n= batas jumlah banyaknya data

    i=1 = batas data dari i = 1

    w*i*= bobot yang diberkan pada atribut ke-i

    Chord distance¶

    ​ Chord distance digunakan untuk mengukur jarak atau citra dan merupakan formula untuk menyempurnakan kelemahan dari formula euclidean distance, dengan formula sebagai berikut, $$ d _ { \text {chord} } = \left ( 2 - 2 \frac { \sum _ { i = 1 } ^ { n } x _ { i } y _ { i } } { | x | _ { 2 } | y | _ { 2 } } \right) ^ { \frac { 1 } { 2 } } $$ Keterangan:

    x*i* = nilai x pada data ke-i

    y*i* = nilai y pada data ke-i

    n= batas jumlah banyaknya data

    i=1 = batas data dari i = 1

    Cosine measure¶

    ​ Cosine similarity digunakan dalam ruang positif, dimana hasilnya dibatasi antara nilai 0 dan 1. Jika nilainya 0 maka dokumen tersebut dikatakan mirip, dan jika hasilnya 1 maka nilai tersebut dikatakan tidak mirip. Misalnya dalam information retrieval, masing-masing kata atau istilah diasumsikan sebagai dimensi yang berbeda dan dokumen ditandai dengan vector dimana nilai masing-masing dimensi sesuai dengan berapa istilah muncul dalam dokumen.

    ​ Metode Cosine Similarity merupakan metode yang digunakan untuk menghitung similarity (tingkat kesamaan) antar dua buah objek. Secara umum penghitungan metode ini didasarkan pada vector space similarity measure. Cosine similarity merupakan metode perhitungan jarak antara vektor d1 dan d2 yang menghasilkan sudut cosine x diantara kedua vektor tersebut. Metode ini digunakan untuk menghitung nilai kedekatan vektor d dengan centroid atau pusat massa pada masing-masing cluster. $$ Cosine(x,y)=\frac { \sum _ { i = 1 } ^ { n } x _ { i } y _ { i } } { | x | _ { 2 } | y | _ { 2 } } $$ x*i* = nilai x pada data ke-i

    y*i* = nilai y pada data ke-i

    n= batas jumlah banyaknya data

    i=1 = batas data dari i = 1

    Pearson corelation¶

    ​ Korelasi Pearson merupakan salah satu ukuran korelasi yang digunakan untuk mengukur kekuatan dan arah hubungan linier dari dua variable. Perlu diingat, bahwa "Koefisien korelasi hanya mengukur kekuatan hubungan linier dan tidak pada hubungan non linier". Dan "Adanya hubungan linier yang kuat di antara variabel tidak selalu berarti ada hubungan kausalitas atau sebab-akibat". Contoh, peneliti ingin melihat apakah terdapat hubungan antara minat mahasiswa terhadap matakuliah Pengantar Ilmu Politik (x) dengan minat mahasiswa terhadap Berpolitik Praktis (y). $$ Pearson ( x , y ) = \frac { \sum _ { i = 1 } ^ { n } ( x _ { i } - \mu _ { x } ) ( y _ { i } - \mu _ { y } ) } { \sqrt { \sum _ { i = 1 } ^ { n } ( x _ { i } - y _ { i } ) ^ { 2 } } \sqrt { \sum _ { i = 1 } ^ { n } ( x _ { i } - y _ { i } ) ^ { 2 } } } $$ Keterangan:

    x*i* = nilai x pada data ke-i

    y*i* = nilai y pada data ke-i

    n= batas jumlah banyaknya data

    i=1 = batas data dari i = 1

    μ*x* = nilai μ pada data ke-x

    μ*y* = nilai μ pada data ke-y

    Mahalanobis distance¶

    ​ Malahanobis distance adalah mengukur jarak antara dua objek, seperti L2 Norm dengan memikirkan korelasi antar objek dalam bentuk vector variabel dari objek dan matrik covariance dari kedua objek tersebut dengan rumus sebagai berikut: $$ d _ { m a h } = \sqrt { ( x - y ) S ^ { - 1 } ( x - y ) ^ { T } } $$ Keterangan:

    x*i* = nilai x pada data ke-i

    y*i* = nilai y pada data ke-i

    n= batas jumlah banyaknya data

    i=1 = batas data dari i = 1

    S^-1= matriks covariance data

    ​ Apabila matrik covariance adalah matrik identity maka mahalanobis distance adalah Euclidean distance. Dan jika matrik covariance adalah matrik diagonal, maka mahalanobis distance adalah Normalised Euclidean distance dimana korelasi antara objek dianggap tidak ada.

    B. Mengukur Jarak Atribut Binary¶

    ​ Pada atribut biner simetris, setiap status memiliki nilai yang sama. Sedangkan pada atribut asimetris, setiap status memiliki nilai yang berbeda. Dissimilarity antara objek data i dan objek data j pada atribut biner simetris sebagai berikut, $$ d\left ( i,j \right )= \frac{r+s}{q+r+s+t} $$ Sedangkan pada biner asimetris, formulanya sebagai berikut, $$ d\left ( i,j \right )= \frac{r+s}{q+r+s} $$ Keterangan:

    q adalah jumlah atribut yang bernilai 1 untuk kedua objek;

    r adalah jumlah atribut yang bernilai 1 untuk objek i dan bernilai 0 untuk objek j;

    s adalah jumlah atribut yang bernilai 0 untuk objek i dan bernilai 1 untuk objek j;

    t adalah jumlah atribut yang bernilai 0 untuk kedua objek. Jumlah dari 4 variabel tersebut sama dengan jumlah atribut p, p = q+r+s+t.

    C. Mengukur Jarak Tipe Categorical¶

    ​ Atribut categorical disebut juga atribut nominal, karena nilainya menggambarkan kategori , status yang tidak memiliki urutan atau tingkatan. Contoh jarak untuk atribut nominal, yaitu atribut kategori pelanggan yang memiliki 3 status, yaitu Silver,Gold, atau Platinum, dinyatakan sebagai M=3 (M adalah jumlah status (nilai) dalam suatuatribut nominal). Dissimilaraty antara objek i dengan objek j dihitung berdasarkan rasio ketidaksamaan status yang secara matematis dinyatakan sebagai d(i,j) = (p-m)/p dimana m menyatakan jumlah status yang bernilai sama dan p adalah jumlah atribut pada objek tersebut.

    Ovarlay Metric¶

    ​ Ovarlay metric kurang begitu efektif digunakan untuk mengukur jarak, karena kurang memanfaatkan adanya informasi yang telah ada, $$ d ( x , y ) = \sum _ { i = 1 } ^ { n } \delta ( a _ { i } ( x ) , a _ { i } ( y ) ) $$ Keterangan:

    ai(x) dan ai(y) = nilai atribut pada data ke-i yaitu Ai

    n= batas jumlah banyaknya data

    i=1 = batas data dari i = 1

    Value Difference Metric (VDM)¶

    ​ Value Difference Metric (VDM) diperkenalkan oleh Stanfill dan Waltz pada tahun 1986 untuk memberikan fungsi perhitungan jarak yang tepat untuk data dengan atribut kategorikal. VDM mendefinisikan jarak antara dua valuex dan y dari atribut sebagai berikut: $$ d ( x , y ) = \sum _ { i = 1 } ^ { n } \sum _ { c = 1 } ^ { C } \left| P ( c | a _ { i } ( x ) ) - P ( c | a _ { i } ( y ) ) \right | $$ Keterangan:

    C= banyaknya kelas

    P(c|ai(x))P(c|ai(x)) =probabilitas bersyarat

    P(c|x)P(c|x) dan P(c|y)P(c|y)=probabilitas keanggotaan kelas

    n= batas jumlah banyaknya data

    i=1 = batas data dari i = 1

    Yang kemudian dikembangkan oleh Myles dan Hand dan didefinisikan dengan rumus sebagai berikut: $$ d ( x , y ) = \sum _ { c = 1 } ^ { C } \left | P ( c | x ) - P ( c | y ) \right| $$ ​ Dengan menggunakan metode perhitungan jarak VDM jarak antara dua value (x,y) dianggap lebih dekat jika memiliki tingkat kesamaan (similarity) klasifikasi yang lebih tinggi. Kelemahan dari metode perhitungan jarak VDM ini ialah jarak antara dua value akan dianggap 0 ketika value yang muncul dalam vektor inputan baru tidak terdapat pada training set. Jika value x tidak pernah muncul dalam atribut, maka nilai untuk setiap C (kelas) akan 0 dan juga akan bernilai 0. Fungsi jarak VDM ini tidak tepat digunakan langsung pada data dengan atribut kontinue. Jika metode VDM digunakan langsung pada atribut kontinue, semua value dapat bernilai unik. Nilai dapat bernlai 1 untuk semua value x dan dapat bernilai 1 untuk kelas 1 (C=1) dan bernilai 0 untuk kelas lainnya. Vektor inputan baru cenderung memiliki value yang unik, sehingga perhitungan jarak dapat menghasilkan nilai 0 dan nilai yang dihasilkan menjadi tidak berguna, oleh karena itu metode perhitungan jarak ini tidak tepat diaplikasikan untuk mengukur jarak antar data dengan atribut kontinue. (Wilson : 1997)

    Minimum Risk Metric (MRM)¶

    ​ Minimum risk metric digunakan untuk meminumkan kesalahan-kesalahan yang akan terjadi pada suatu data. Dimana jarak adalah hasil dari jumlah probabilitas keanggotaan kelas x dikurangi probabilitas keanggotaan y dengan batas data c=1 sampai batas jumlah banyaknya data. $$ d ( x , y ) = \sum _ { c = 1 } ^ { C } P ( c | x ) ( 1 - P ( c | y ) ) $$ Keterangan:

    P(c|x)P(c|x) dan P(c|y)P(c|y)=probabilitas keanggotaan kelas

    c= batas jumlah banyaknya data

    c=1 = batas data dari c= 1

    D. Mengukur Jarak Tipe Ordinal¶

    ​ Atribut Ordinal Memiliki nilai yang menggambarkan urutan atau peringkat, namun perbedaan ukuran antara dua nilai yang berurutan tidak diketahui. Tipe ini biasanya digunakan dalam survey untuk penilaian subjektif (kualitatif) yang tidak dapat diukur secara objektif. Misalkan survey tentang kepuasan pelanggan yang menghasilakan atribut bernilai ordinal, yaitu 0, 1, 2, dan 3. $$ z _ { i f } = \frac { r _ { i f } - 1 } { M _ { f } - 1 } $$ Atribut tipe ordinal diperlakukan sebagai interval-scaled , dimana:

    a. xif diganti dengan peringkat r*if* anggota{1,.....,M*f*}

    b. Petakan ke [0, 1] dengan mengganti objek ke i dan variabel ke f dengan

    c. Cara penghitungannya sama dengan interval variable.

    E. Mengukur Jarak Tipe Campuran¶

    ​ Mengukur jarak menggunakan tipe campuran yaitu mengukur jarakdimana didalamnya mencakup beberapa tipe atribut , yaitu atribut numerik, atribut binary (binary simetris dan binary asimetris), atribut categorical dan atribut ordinal. Teknik ini digunakan untuk menggabungkan beberapa atribut yang berbeda menjadi satu formula untuk menghitung suatu jarak pada suatu data. Pada tipe campuran ini menggunakan formula weighted untuk dapat mengkombinasikan semua formula yang berbeda, formulanya sebagai berikut: $$ d ( i , j ) = \frac { \sum _ { f = 1 } ^ { p } \delta _ { i j } ^ { ( f ) } d _ { i j } ^ { ( f ) } } { \sum _ { f = 1 } ^ { p } \delta _ { i j } ^ { ( f ) } } $$

    Atribut-Atribut dalam Data Mining¶

    ​ Atribut adalah karakteristik atau ciri atau variabel yang mendeskripsikan suatu objek. Sedangkan objek adalah sesuatu yang digambarkan oleh atribut. Berikut tipe-tipe atribut dalam data mining:

    A. Nominal

    ​ Nominal adalah tipe atribut yang tidak mengenal urutan, contoh:

    • Warna baju : pink. putih, hitam, biru, dan lainnya.
    • Suku bangsa : Ambon, Batak, Jawa, dan lainnya.

    B. Ordinal

    ​ Ordinal adalah tipe atribut yang mengenal urutan atau mengidentifikasikan informasi mengenai suatu tingkatan yang menjadikan pembeda dari objek. Contoh:

    • Peringkat : juara 1, juara 2, juara 3
    • Ukuran baju : S, M, L, XL, XXL, XXXL

    C. Biner

    ​ Biner adalah atribut nominal diamana hanya memiliki dua kategori yaitu 1 atau 0. Dimana 0 biasanya berarti tidak ada (False), dan 1 berarti ada (True). Contoh:

    • Jenis kelamin : Laki-laki dan Perempuan
    • Memiliki rumah sendiri : Ya dan Tidak
    • Power : On dan Off

    D. Numerik

    ​ Numerik adalah atribut kuantitatif yang terukur, biasanya dalam bentuk tipe integer. Contoh:

    • Suhu : 50 derajat celcius, 30 derajat celcius, dan lainnya,
    • Tahun : 2000, 2019, 1999, dan lainnya.

def chordDist(v1,v2,jnis):
    jmlh=0
    normv1=0
    normv2=0
    for x in range (len(jnis)):
        normv1=normv1+(int(k.values.tolist()[v1][jnis[x]])**2)

    return ((2-(2*jmlh/((normv1**0.5)*(normv2**0.5)))**0.5))

def binaryDist(v1,v2,jnis):
    q=0
    r=0
    s=0
    t=0
    for x in range (len(jnis)):
        if (int(k.values.tolist()[v1][jnis[x]]))==1 and (int(k.values.tolist()[v2][jnis[x]]))==1:
            q=q+1
        elif (int(k.values.tolist()[v1][jnis[x]]))==1 and (int(k.values.tolist()[v2][jnis[x]]))==2:
            r=r+1
        elif (int(k.values.tolist()[v1][jnis[x]]))==2 and (int(k.values.tolist()[v2][jnis[x]]))==1:
            s=s+1
        else:
            t=t+1
    return ((r+s)/(q+r+s+t))
